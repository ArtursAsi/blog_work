### Instructions:
Clone project
>git clone https://github.com/ArtursAsi/blog_work

Install Composer
>composer install

Install npm
>npm install

Create .env file
>cp .env.example .env

Create database
>link in .env file

Mailtrap.io is required
>link your mailtrap.io account in .env file

Link storage 
>php artisan Storage:link

Migrate Database
>php artisan migrate

Generate key
>php artisan key:generate

Run app 
>php artisan serve
